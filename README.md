# vue-electron-app

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn electron:serve
```

### Compiles and minifies for production

```
yarn electron:build

// build for all platforms
yarn electron:build -mwl
```

### Dependecies

- Vue Router 4

- vuetify 3

```html
https://next.vuetifyjs.com/
```

- bootstrap vue
